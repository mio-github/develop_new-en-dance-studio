# En Dance Studio ERPシステム

## プロジェクト概要

En Dance Studioは日本有数のダンススタジオで、このプロジェクトはエン株式会社のためのダンススタジオ管理ERPシステムの刷新を目的としています。Mio Systemが手掛けるこのプロジェクトでは、最新のAI技術を活用し、モダンで使いやすいシステムの構築を目指しています。

## 目的

- 既存の日本のシステムをフルスクラッチで再構築
- アメリカの「MindBody」システムの優れた機能を取り入れ
- AI技術を活用した業務効率化と顧客体験の向上

## ディレクトリ構造

- `now_JP_system/` - 現在の日本のシステムのスクリーンショット
- `now_USA_mind_body/` - アメリカのMindBodyシステムのスクリーンショット
- `mio_desgin_system/` - Mioのデザインシステム（開発中）

## 主要機能（計画中）

- 会員管理
- スケジュール管理
- 決済・会計管理
- マーケティング・コミュニケーション
- レポート・分析
- インストラクター管理
- 在庫・物品管理
- モバイル対応

## 技術スタック（予定）

- フロントエンド: Next.js, React, TypeScript
- バックエンド: Node.js, Express
- データベース: PostgreSQL
- 認証: OAuth 2.0, JWT
- デプロイ: Vercel, AWS
